# Damien's Sunday Revenue Reports Dashboard

Automated revenue analysis dashboard for Brightwild property unit groups.

## Features

- **5 Unit Groups**: KHE Original 10, Gemma 4 Units, Hotel Cabana HC 50, 720 D Simonton, Kamp 430 Fleming
- **3 Data Points**: CY (Current Year), STLY (Same Time Last Year), LY Final
- **Auto-Updates**: Every Sunday at 8:00 AM ET via BigQuery data pull
- **Interactive Charts**: Built with Chart.js for responsive visualization
- **Professional Dashboard**: Clean, modern UI with statistics overview

## Data Sources

- **BigQuery**: `brightwild_analytics_mart.unit_calendar`
- **Revenue Field**: `assignment_night_total_revenue`
- **Date Logic**: 
  - CY: Current year aggregated monthly revenue
  - STLY: Revenue as of same day last year (today - 364 days)
  - LY Final: Actual final revenue for comparison months

## Unit Group Definitions

1. **720 D Simonton**: Units '720 D Simonton Front', '720 D Simonton Rear'
2. **Gemma 4 Units**: Units 'Gemma 19', 'Gemma 21', 'Gemma 25', 'Gemma 27'
3. **Hotel Cabana HC 50**: Units starting with 'HC 50'
4. **Kamp 430 Fleming**: All units in complex_2_name = '430 Fleming'
5. **KHE Original 10**: Specific 10 KHE units (2,4,6,8,10,12,14,16 Shoreline + 2,4 Terrace)

## Deployment

This dashboard is deployed via:
1. **GitHub Repository**: Source code and data
2. **Vercel**: Automatic deployment from GitHub
3. **Cron Automation**: Sunday 8 AM ET data updates

## Technical Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Charts**: Chart.js
- **Data**: JSON file updated via BigQuery
- **Hosting**: Vercel
- **Automation**: OpenClaw cron jobs
